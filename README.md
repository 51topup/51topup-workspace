# Stack Stack

- Java 21 with virtual thread enabled
- Spring Boot 3.4.0
- Vaadin 24.5
- jOOQ 3.19.15
- PostgreSQL 17
- Redis 7
- Nats Server 2.11

- Karibu-DSL: https://github.com/mvysny/karibu-dsl

- jOOQ Spring Integration: https://github.com/martinellich/jooq-spring
